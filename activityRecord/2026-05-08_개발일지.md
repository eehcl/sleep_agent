## 📅 오늘 진행 사항
### 1. 사용자 경험(UX) 및 대화 로직 고도화 기획
- **자연어 트리거 도입:** `/start` 명령어 의존도를 낮추고 '잘래', '졸려', '피곤하다' 등 사용자의 일상적인 수면 의도 표현에 반응하도록 설계 변경 예정
- **모호한 표현 분석 기획:** "한 잔 정도", "절반쯤 마심", "자세히는 모름" 등 사용자의 주관적이고 애매한 답변을 Gemma가 상식적으로 판단하여 수치화(Data Parsing)하는 로직 구상
- **개인 비서 페르소나 강화:** 단순 데이터 수집을 넘어 사용자의 말에 공감하고 위로를 건네는 다정한 '수면 비서' 컨셉으로 말투 및 인터페이스 개선 방향 확립

### 2. 데이터 분석 기능 강화 설계
- **날짜 기반 데이터 통합:** 하루에 여러 번 대화를 나누더라도 당일 날짜(YYYY-MM-DD)를 기준으로 데이터를 합산하거나 업데이트하여 일관된 리포트를 제공하도록 DB 로직 설계
- **지능형 카페인 컨설팅:** 과거 섭취 데이터를 기반으로 "오늘은 평소보다 많이 드셨네요", "몇 시 이후에는 섭취를 제한하세요" 등 선제적인 가이드 제공 기획

### 3. 차세대 '브레인 주도형' 아키텍처 구상
- **State Machine 탈피:** 정해진 질문 순서(Fatigue -> Stress)를 따르지 않고, Gemma가 대화 맥락에서 정보를 추출하며 부족한 부분만 골라 물어보는 자유 대화형(Free-form) 구조로의 전환 계획 수립
