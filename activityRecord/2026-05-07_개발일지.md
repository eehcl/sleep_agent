### 1. Gemma 4 e2b 추천 로직
사용자의 상태를 분석하여 맞춤형 수면 가이드를 생성하는 핵심 함수입니다.
```python
async def get_recommendation_from_gemma(user_state: dict) -> str:
    """Gemma 모델에 사용자 상태를 전달하고 수면 콘텐츠 추천을 받는다."""
    prompt = (
        "당신은 사용자의 컨디션을 분석해 수면에 도움이 되는 콘텐츠를 추천하는 전문가입니다.\n\n"
        "다음은 사용자의 현재 상태입니다:\n"
        f"- 피곤함 정도(1~5): {user_state['fatigue']}\n"
        f"- 스트레스 정도(1~5): {user_state['stress']}\n"
        f"- 오늘 카페인 섭취: {user_state['caffeine']}\n"
        f"- 현재 기분: {user_state['mood']}\n"
        f"- 목표 취침 시간: {user_state['target_time']}\n"
        f"- 내일 기상 시간: {user_state['wake_up_time']}\n\n"
        "이 상태에 가장 잘 어울리는 수면 콘텐츠를 추천하고 그 이유를 과학적으로 설명해 줘. "
        "또한 유튜브에서 검색하면 좋을 최적의 검색어도 한 문장으로 추천해 줘."
    )

    client = ollama.AsyncClient()
    response = await client.chat(
        model="gemma4:e2b",
        messages=[{"role": "user", "content": prompt}],
    )
    return response["message"]["content"]
```
